FORMULAS:

bat_ab = bat_pa - bat_bb - bat_hbp - (bat_scb+bat_sf) - bat_ci
bat_hits = bat_bunt_singles+bat_1b+bat_2b+bat_3b+bat_hr
bat_xbh = bat_2b+bat_3b+bat_hr
bat_tb = bat_bunt_singles+bat_1b+(2*bat_2b)+(3*bat_3b)+(4*bat_hr)
bat_ob = bat_bunt_singles + bat_1b + bat_2b + bat_3b + bat_hr + bat_bb + bat_hbp
bat_runs_created = (bat_hits + bat_bb + bat_hbp - bat_cs - bat_gidp) * (bat_tb + (.26*(bat_bb - bat_bbi + bat_hbp)) + (.52*(bat_scb + bat_sf + bat_sb))) /(bat_ab + bat_bb + bat_hbp + bat_scb + bat_sf)
bat_avg = bat_hits/bat_ab
bat_strikeouts = bat_ko_looking+bat_ko_swinging
bat_bb_per_k = bat_bb/bat_strikeouts
bat_bb_per_pa = bat_bb/bat_pa
bat_sac = bat_scb+bat_sf
bat_rpa = (bat_runs + bat_rbi - bat_hr)/bat_pa
bat_obp = bat_ob/(bat_ab + bat_bb + bat_hbp + bat_sf)
bat_obpe = (bat_ob+bat_roe)/(bat_ab + bat_bb + bat_hbp + bat_sf)

StatNotes
Quality at Bats
(QAB)		 Quality at Bats are a statistics that has become popular to talk about, but there is not a hard and fast definition for what makes up a QAB. We have decided to create a few variants and allow you to customize the QAB value to what you think is important. The following tables show our defaults:

QAB-1
  • Base Hit
  • Sac Bunt
  • Sac Fly
  • Hard ground ball
  • Hard line drive
  • Hard fly ball
  • 6+ pitches
 QAB-2
  • Base Hit
  • RBI
  • Sac Bunt
  • Sac Fly
  • 6+ Pitches
 QAB-3
  • Hit w/RISP
  • RBI
  • Sac Bunt
  • Sac Fly
  • 6+ Pitches
Game Score
(GSc)		 Game score is described here: http://en.wikipedia.org/wiki/Game_score
  1. Start with 50 points.
  2. Add 1 point for each out recorded, so 3 points for every complete inning pitched.
  3. Add 2 points for each inning completed after the 4th.
  4. Add 1 point for each strikeout.
  5. Subtract 2 points for each hit allowed.
  6. Subtract 4 points for each earned run allowed.
  7. Subtract 2 points for each unearned run allowed.
  8. Subtract 1 point for each walk.
Quality Start
(QS)		 Related to Game Score above, a Quality Start is defined as a Game Score > 50.
Tough Loss
(TL)		 Related to Game Score above, a Tough Loss is a pitcher that gets a Loss but had a Quality Start.
Cheap Wins
(CW)		 Related to Game Score above, a Cheap Win is a pitcher that gets a Win but had a non Quality Start.
Runs Created
(RC)		 We use the "technical" version as described here: http://en.wikipedia.org/wiki/Runs_created
